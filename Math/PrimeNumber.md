**정의**

- 1과 자기 자신으로 밖에 나눠 지지 않는 수
- 자연수 N이 소수 인지 판별하는 법
  - 정의 대로 구하기 -> 느림
    - N % k == 0 인 경우 소수 X (단, 2<=k<=N-1 인 모든 수) 
  - 에레토스테네스 체 -> 빠름
